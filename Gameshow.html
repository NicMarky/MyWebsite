This is a Gameshow Button Buzzer which runs by Arduino Codes.
It consist of 11 Button which indicates 11 players to play and the first player that push the game button will allow to answer
the question which means by default al button lights on unti one player who pushes it first stays it on while the other is off. This is a "First-Response" lockout circuit. The logic ensures that the moment one player presses their button, all other inputs are instantly disabled (locked out). The system also includes a "referee" logic where an incorrect answer resets the lockout, allowing other players back into the game.
