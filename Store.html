This is a Store Inventory System which runs by C++ Code. The system starts with an initial stock of 50 items per product, and whenever a user purchases an item, the program automatically deducts one from the total inventory (e.g., 50 decreases to 49). The inventory display updates in real time to reflect remaining stock, allowing users to monitor available items and simulate a simple purchasing and stock management process.
